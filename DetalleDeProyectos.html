Descripción del Proyecto

Este proyecto tiene como objetivo desarrollar una aplicación móvil innovadora para la gestión de tareas y la colaboración en equipo. La aplicación permitirá a los usuarios crear, asignar y seguir el progreso de las tareas, así como comunicarse y compartir archivos de manera eficiente.
